# 🌾 Crop Yield Prediction using Machine Learning

This project predicts agricultural crop yield based on environmental and geographical factors such as rainfall, temperature, crop type, and location using a Random Forest Regressor.

---

## 📌 Overview

Accurate crop yield prediction helps in better agricultural planning and decision-making. This project integrates multiple datasets (rainfall, temperature, and crop yield) to build a machine learning model that estimates yield effectively.

---

## 🚀 Features

- Data preprocessing and cleaning
- Merging multiple datasets (rainfall, temperature, yield)
- Feature engineering (Area, Crop encoding)
- Machine Learning model using Random Forest Regressor
- Model evaluation using R² Score and MAE
- Feature importance visualization

---

## 🧠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 📂 Dataset

The project uses three datasets:

- Rainfall dataset  
- Temperature dataset  
- Crop Yield dataset  

These datasets are merged using common columns such as **Area** and **Year**.

---

## ⚙️ How It Works

1. Load datasets
2. Clean and preprocess data
3. Merge datasets into a single dataframe
4. Encode categorical variables (Area, Crop)
5. Train-test split
6. Train model using Random Forest Regressor
7. Evaluate model performance

---

## 📊 Model Evaluation

- **R² Score** – Measures model accuracy  
- **Mean Absolute Error (MAE)** – Measures prediction error
